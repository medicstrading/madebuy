# Testing Reviewer

**Domain:** Test coverage, test quality
**Type:** Advisory (non-editing)

---

## Review Checklist

- [ ] Unit tests exist
- [ ] Integration tests exist
- [ ] Edge cases covered
- [ ] Error cases tested
- [ ] Mocks appropriate
- [ ] Tests are maintainable

---

## Output Format

```
## Testing Review: [Feature]

### Coverage
[What's tested vs not]

### Missing Tests
1. [Test needed]

### Recommendations
1. [Improvement]
```
