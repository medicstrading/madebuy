# Security Context Reviewer

**Domain:** Auth, permissions, data protection
**Type:** Advisory (non-editing)

---

## Review Checklist

- [ ] Authentication required
- [ ] Authorization (role-based)
- [ ] Input validation
- [ ] Output encoding
- [ ] Sensitive data handling
- [ ] Rate limiting
- [ ] CORS configuration
