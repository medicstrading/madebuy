# API Integrations Reviewer

**Domain:** External APIs, suppliers, social media
**Type:** Advisory (non-editing)

---

## Review Checklist

- [ ] Rate limiting handled
- [ ] Retry logic with backoff
- [ ] Circuit breaker pattern
- [ ] Error handling
- [ ] Response caching
- [ ] API versioning
- [ ] Credential management
